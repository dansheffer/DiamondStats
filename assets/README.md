# Assets Directory

## Player Placeholder Image

You need to add a `player-placeholder.png` file to this directory for the PlayerAvatar component fallback.

**Suggested placeholder image:**
- A simple silhouette or baseball player icon
- Size: 200x200 pixels minimum
- Format: PNG with transparent background preferred
- Neutral/minimalist design

You can create one using any image editor or download a free baseball player silhouette icon from sites like:
- Flaticon
- Icons8
- The Noun Project

Place the file at: `assets/player-placeholder.png`
