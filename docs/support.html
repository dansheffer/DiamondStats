Having an Issue?

If you're experiencing a problem with Diamond Stats, try these steps first:

  • Pull down on any screen to refresh the data.
  • Make sure you have an internet connection — the app needs to reach the MLB Stats API to load scores and stats.
  • Force-quit the app and reopen it.
  • Check for app updates in the App Store.
